PrecedenceParse
===============

<:PrecedenceParse:> is an analysis/rewrite pass for the <:AST:>
<:IntermediateLanguage:>, invoked from <:Elaborate:>.

== Description ==

This pass rewrites <:AST:> function clauses, expressions, and patterns
to resolve <:OperatorPrecedence:>.
