ToRSSA
======

<:ToRSSA:> is a translation pass from the <:SSA2:>
<:IntermediateLanguage:> to the <:RSSA:> <:IntermediateLanguage:>.

== Description ==

This pass converts a <:SSA2:> program into a <:RSSA:> program.

It uses <:PackedRepresentation:>.
