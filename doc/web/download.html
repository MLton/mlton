The latest stable version of MLton, 20020923, is available in the following formats.

  • Binary
    • rpm for Red Hat 7.x (glibc 2.2, rpm 4.x, gmp 3.1.1, gmp-devel 3.1.1)
    • deb for Debian GNU/Linux 3.0 (libc6 >= 2.2.4-4, libgmp3-dev)
    • tgz for Linux (glibc 2.2)
    • tgz for FreeBSD
    • tgz for Cygwin/Windows
  • Source
    • tgz for any platform
    • srpm for Red Hat 7.x (rpm 4.x)
You can also access the MLton sources in the following ways.
  • on the web with ViewCVS
  • via anonymous CVS with
    cvs -d :pserver:anonymous@cvs.mlton.org:/cvsroot/mlton co mlton
You can also download older and newer, experimental versions of MLton.
You can also download other MLton packages for Debian GNU/Linux.

Here are the shared library requirements for the Linux binary versions of MLton. 

% rpm -qp --requires mlton-20020923-1.i386.rpm 
rpmlib(PayloadFilesHavePrefix) <= 4.0-1

libc.so.6  
libm.so.6  
/bin/sh  
libc.so.6(GLIBC_2.0)  
libc.so.6(GLIBC_2.1)  
libc.so.6(GLIBC_2.1.3)  
libm.so.6(GLIBC_2.0)
